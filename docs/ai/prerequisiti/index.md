# Prerequisiti matematici

Le idee dell'AI poggiano su tre pilastri matematici molto concreti:

1. **Vettori e matrici** per rappresentare dati e parametri.
2. **Trasformazioni lineari** per capire cosa fa "moltiplicare per una matrice".
3. **Distribuzione gaussiana** per ragionare sull'incertezza nei dati e sulle stime.
