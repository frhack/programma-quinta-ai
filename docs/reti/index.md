# Reti

Architettura a livelli, indirizzamento IPv4, subnetting, protocolli principali e laboratorio.

## Argomenti

- Architettura a livelli, modello TCP/IP e ISO/OSI
- Indirizzamento IPv4: classi, netmask, indirizzo di rete e di broadcast
- Subnetting: calcolo sottoreti, VLSM
- Protocolli principali: IP, TCP/UDP, DNS, DHCP, HTTP
- Comandi diagnostici: `ping`, `ipconfig`/`ifconfig`, `tracert`
